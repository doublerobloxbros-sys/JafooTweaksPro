// FAQ

Frequently Asked

What is Jafoo Tweaks? ▾
Jafoo Tweaks is a free, open-source Windows gaming optimizer built with PowerShell and an Electron UI. It applies 90+ tested registry, network, CPU, GPU, and storage tweaks to maximize your gaming performance.
Is it safe? Will it break my PC? ▾
Yes. Jafoo Tweaks automatically backs up your registry before applying any changes. Every tweak is reversible. The source code is fully open on GitHub — you can inspect every line before running it.
Is it actually free? What's the catch? ▾
No catch. Jafoo Tweaks is 100% free, no account required, no telemetry, no ads. It's a passion project built for the gaming community.
What Windows versions does it support? ▾
Jafoo Tweaks supports Windows 10 and Windows 11. It is built with PowerShell 5.1 compatibility so it works on all modern Windows versions without needing any updates or additional runtimes.
How much FPS improvement can I expect? ▾
Results vary depending on your hardware and how bloated your Windows install is. Most users see meaningful FPS gains and more consistent frame times, especially after debloating and applying game-specific profiles. Use the built-in latency tester to benchmark before and after.
Does it work with anti-cheat (Vanguard, EAC, BattlEye)? ▾
Yes. Jafoo Tweaks applies OS-level and registry optimizations — it does not modify any game files, inject into processes, or bypass anti-cheat systems. It's compatible with Vanguard, Easy Anti-Cheat, and BattlEye.
Where can I get support or report a bug? ▾
Open an issue on the GitHub repository at github.com/doublerobloxbros-sys/JafooTweaks, or reach out on Twitter/X @JaydenBoxedY0U.
